# dotfiles

Arch Linux / Hyprland setup managed with [chezmoi](https://chezmoi.io).

## Stack

| Category | Tool |
|---|---|
| Compositor | Hyprland |
| Status bar | Waybar |
| Terminal | Kitty |
| Multiplexer | Tmux |
| Editor | Neovim |
| Launcher | Rofi |
| Notifications | Dunst |
| File manager | Yazi |
| Theming | Matugen (Material You) |
| Wallpaper | awww |
| Audio viz | Cava |
| Music client | SubTUI |

---

## Fresh install

```bash
curl -fsSL https://raw.githubusercontent.com/Kai-Doh/dotfiles/main/install.sh | bash
```

This will:
1. Install chezmoi and all packages via `yay`
2. Prompt once for machine-specific values (monitor name, backlight device, timezone, Firefox profile)
3. Apply all configs automatically
4. Bootstrap tmux plugins, enable systemd services

After install, log into Hyprland then:
```bash
nvim +Lazy sync       # install neovim plugins
# in tmux: prefix + I  # install tmux plugins
```

## Updating

Pull latest changes and re-apply on any machine:
```bash
update-dot
```

---

## Adding a new app config

```bash
chezmoi add ~/.config/newapp
chezmoi cd
git add . && git commit -m "add newapp config" && git push
```

If the app needs a package, add it to `packages.txt` before committing.

## Theming

Everything is themed via [matugen](https://github.com/InioX/matugen). To change the theme:
```bash
wallpaper ~/Pictures/Wallpapers/yourimage.jpg
```

This regenerates colors for Hyprland, Waybar, Kitty, Neovim, Rofi, Dunst, Cava, Yazi, GTK, and Firefox in one shot.

## Machine-specific config

On first install, chezmoi prompts for:

| Value | What it's for | Default |
|---|---|---|
| Monitor name | `hyprpaper.conf` | `eDP-1` |
| Backlight device | Waybar brightness module | `intel_backlight` |
| Timezone | Waybar clock | `Europe/Zurich` |
| Firefox profile | Matugen Firefox theme output | auto-detected |

These are saved to `~/.config/chezmoi/chezmoi.toml` — never committed.  
To find your values: `hyprctl monitors` / `ls /sys/class/backlight/`

## Security

The following are never committed:
- `~/.config/subtui/credentials.toml` — copy from `credentials.toml.example` and fill in
- `~/.config/sunshine/credentials/` — auto-generated by Sunshine
- Browser profiles (Firefox, Zen)
- Claude Code local settings
